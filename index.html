Chord System Demo

Interactive Chord Selector

Select a chord and see it displayed on the keyboard.

Standalone Keyboard

A standalone piano keyboard component.

How to Use

Include the chord-system-bundle.js file in your HTML:

<script src="js/chord-system-bundle.js"></script>

Initialize a chord selector:

const chordSelector = ChordSystem.init('#selector-container');

Create a standalone keyboard:

const keyboard = ChordSystem.createKeyboard('#keyboard-container');

Highlight notes on a keyboard:

// MIDI note numbers for a C major chord keyboard.highlightChordNotes([60, 64, 67]);
